# Heart Rate Monitoring System (Arduino)

## Overview
This project is a simple heart rate monitoring system built using Arduino. It reads pulse signals from a sensor and processes them to estimate heart rate.

## Components Used
- Arduino Board
- Heart Rate Sensor
- Jumper Wires
- Breadboard

## Working Principle
The sensor detects pulse signals and sends them to the Arduino. The Arduino processes the signal and calculates the heart rate.

## Applications
- Basic health monitoring
- Learning biomedical signal basics
